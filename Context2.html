The Engineering Layer

Idempotent Sync Pattern

-- ENSURING DATA CONSISTENCY ACROSS OFFLINE SYNC
MERGE Logistics.Fact_Transport AS Target
USING Staging.Mobile_Upload AS Source
ON Target.Sync_Hash = Source.Sync_Hash
WHEN MATCHED AND Source.Last_Updated > Target.Last_Updated THEN
    UPDATE SET 
        Target.Distance_KM = Source.Distance_KM,
        Target.Fuel_Used = Source.Fuel_Used,
        Target.Status = 'VERIFIED'
WHEN NOT MATCHED THEN
    INSERT (Sync_Hash, Vehicle_ID, Driver_ID, Distance_KM)
    VALUES (Source.Sync_Hash, Source.Vehicle_ID, Source.Driver_ID, Source.Distance_KM);

Prevents duplicate records from intermittent field connectivity syncs.
